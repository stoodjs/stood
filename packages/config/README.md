`@stood/config`

Configuration files for the `@stood/*` packages.

## Installation

```bash
bun add @stood/config
```

## Usage

### Biome

In your `biome.json`, add the following:

```json
{
	"extends": ["@stood/config/biome"]
}
```

### TypeScript

In your `tsconfig.json`, add the following:

```json
{
	"extends": ["@stood/config/tsconfig"]
}
```
